# сайт Высшего Образования (онлайн)
Современный стек (HTML5 & CSS3). 
Методология БЭМ.
Адаптивность и Кроссбраузерность. Использование гибких сеток (Flexbox)
Интерактивный контент.

    Аккордеоны и Табуляция: Сложные массивы данных и блоки «Вопрос-ответ» организованы с помощью компактных скриптов, что экономит место и улучшает юзабилити.
    Модальная регистрация: Всплывающая форма реализована с фокусом на UX — она легкая, доступная и не перегружает интерфейс.
